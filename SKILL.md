## Core Frameworks & Mental Models

### 1) Intent-First SEO
Use **intent taxonomy** (navigational, informational, transactional, local) before writing or optimizing pages. Prefer intent-fit over volume-chasing because it raises conversion quality and reduces mismatch signals.

### 2) Crawl → Index → Rank Diagnostics
Treat SEO failures as pipeline failures:
1. **Crawlability** — can bots reach content?
2. **Indexability** — can engines keep the right URLs?
3. **Rankability** — do pages have sufficient relevance + authority?

Always debug in this order to avoid false fixes.

### 3) Keyword Clusters, Not Isolated Terms
Group queries by shared intent and expected page type. Map one primary page per cluster to prevent cannibalization. Build internal links around cluster hubs.

### 4) SEO as an Operating System
Run SEO as a repeatable system with clear owners:
- technical health
- content production and refresh
- authority/link growth
- measurement and experimentation

Use quarterly roadmap planning with weekly execution loops.

### 5) Technical Baseline Before Scale
Before content scale, enforce baseline quality:
- clean indexing controls (canonical, robots, noindex where needed)
- stable information architecture
- strong internal linking
- mobile-first UX and performance thresholds
- schema where it improves entity understanding

### 6) Measurement That Explains Causality
Avoid vanity reporting. Build a KPI tree connecting visibility → clicks → sessions → leads/revenue. Segment by page type, intent, device, and cohorts. Keep an experiment ledger with baseline and post-change readouts.

### 7) Authority via Value, Not Manipulation
Prioritize link-worthy assets (original research, tools, deep resources) and digital PR workflows. Value topical relevance and trust over raw link counts. Avoid schemes that increase penalty risk.

### 8) Update/Penalty Resilience
For algorithm updates or manual actions:
- isolate impacted cohorts
- form evidence-based hypotheses
- prioritize high-impact remediations
- measure recovery lag

No panic edits. Controlled iteration beats bulk random change.

### 9) Vertical SEO Playbooks
Run dedicated playbooks for local, mobile, video, image, and other SERP surfaces. Different surfaces require different metadata, asset formats, and trust signals.

### 10) AI-Augmented, Human-Verified SEO
Use generative AI for speed (briefs, drafts, clustering, ideation). Keep human editorial control for factual accuracy, originality, and strategic fit. Prompt quality matters; verification is mandatory.

### 11) Privacy & Regulatory Awareness
Design measurement and data usage with consent and compliance constraints in mind. Durable SEO performance depends on trust, legal alignment, and sustainable data practices.

### 12) Continuous Learning Loop
SEO changes constantly. Institutionalize research, tests, and documentation so team capability compounds over time.
