# Codex Marketplace Package

## Install

```bash
git clone https://github.com/lemonchikHere/art-of-seo-playbook-skill.git
mkdir -p ~/.codex/skills
cp -R art-of-seo-playbook-skill ~/.codex/skills/art-of-seo-playbook
```

## Run

```bash
/art-of-seo-playbook
/art-of-seo-playbook ch07
/art-of-seo-playbook keyword clustering
```
