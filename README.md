## ✨ Why this skill exists

Most SEO docs are either too abstract or too tactical.  
This skill gives you an **operator-grade system**:

- **Intent → Architecture → Content → Authority → Measurement**
- Chapter-by-chapter implementation guidance
- Fast diagnostics for traffic drops and update recovery
- Repeatable operating patterns your team can standardize
