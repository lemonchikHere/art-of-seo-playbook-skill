# Glossary

**Algorithm Update** — изменение алгоритмов ранжирования, влияющее на видимость (Ch 9)
**Anchor Text** — текст ссылки, помогающий интерпретировать целевую страницу (Ch 11)
**Canonical URL** — предпочтительная версия страницы для индексации (Ch 3)
**Content Cluster** — группа связанных материалов вокруг общего интента (Ch 6)
**Core Web Vitals** — метрики UX-производительности (Ch 7)
**Crawl Budget** — условный лимит обхода сайта ботом (Ch 3)
**Ecommerce SEO** — оптимизация под коммерческие сценарии поиска (Ch 1)
**Entity SEO** — оптимизация сущностей и связей в графе знаний (Ch 15)
**Generative AI** — модели, генерирующие текст/медиа для SEO-задач (Ch 2)
**Google Business Profile** — локальная сущность бизнеса в экосистеме Google (Ch 12)
**Index Bloat** — избыточная индексация страниц без ценности (Ch 10)
**Intent** — цель пользователя в запросе (Ch 1)
**Internal Linking** — перелинковка страниц внутри сайта (Ch 7)
**Keyword Difficulty** — относительная сложность ранжирования по запросу (Ch 6)
**Knowledge Graph** — граф сущностей и отношений в поисковой системе (Ch 3)
**Local Pack** — блок локальных результатов в SERP (Ch 12)
**Manual Action** — ручная санкция поисковой системы (Ch 9)
**Organic CTR** — кликабельность органических результатов (Ch 8)
**Prompt Engineering** — проектирование инструкций для LLM (Ch 2)
**Query Intent** — намерение, стоящее за конкретным запросом (Ch 1)
**Ranking Factors** — сигналы, используемые для ранжирования (Ch 3)
**Redirect Chain** — последовательность перенаправлений, ухудшающая краулинг/UX (Ch 10)
**Referring Domain** — домен, ссылающийся на сайт (Ch 11)
**Schema Markup** — структурированная разметка для лучшего понимания контента (Ch 7)
**Search Console** — интерфейс мониторинга видимости и индексации (Ch 4)
**SERP Features** — элементы выдачи кроме классических ссылок (Ch 1)
**Technical SEO** — технические аспекты доступности/индексации/производительности (Ch 7)
**Topic Authority** — доверие к сайту в конкретной тематике (Ch 11)
**Vertical SEO** — оптимизация для специальных вертикалей поиска (Ch 12)
