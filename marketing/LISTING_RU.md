# Тексты листинга (RU)

## Название
Art of SEO Playbook — Codex Skill

## Коротко
Продакшн-система SEO: intent-first подход, диагностика и playbook’и роста.

## Краткое описание
Практичный SEO skill-пакет из 15 глав: чеклисты, метрики, анти-паттерны, сценарии восстановления и рабочие шаблоны.

## Полное описание
Art of SEO Playbook помогает командам делать SEO системно и предсказуемо:

- Планирование от интента, а не от отдельных ключей
- Диагностика по модели Crawl → Index → Rank
- Связка technical/content/links
- Сценарии после algorithm updates и manual actions
- Измерение результата через KPI-циклы

Внутри:
- SKILL.md (главный операционный слой)
- 15 глав с implementation playbook
- glossary + patterns + cheatsheet
- готовые пакеты для Codex / Claude Code / GPT Custom
